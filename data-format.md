# 今彩539開獎資料格式規範

## JSON檔案結構

```json
{
  "last_updated": "2025-06-15T15:30:47.111279",
  "total_records": 500,
  "data": [
    {
      "date": "2025/06/14",
      "numbers": [2, 15, 17, 23, 37],
      "timestamp": "2025-06-14T00:00:00"
    }
  ]
}
```

## 欄位說明

### 根層級
- `last_updated`: 資料最後更新時間（ISO 8601格式）
- `total_records`: 總記錄數
- `data`: 開獎記錄陣列

### 開獎記錄格式
- `date`: 開獎日期（YYYY/MM/DD格式）
- `numbers`: 開獎號碼陣列，包含5個整數（1-39）
- `timestamp`: 開獎日期的ISO 8601時間戳

## 資料規則

1. 開獎號碼範圍：1-39
2. 每期開出5個不重複號碼
3. 號碼按照官方公布順序排列（不排序）
4. 日期按照最新到最舊排序
5. 每頁包含約100筆記錄
